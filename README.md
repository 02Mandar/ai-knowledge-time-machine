# AI Knowledge Time Machine

Built an AI-powered system to analyze how code evolves over time using semantic comparison and Generative AI concepts.

## Features
- Code Evolution Analysis
- Semantic Change Detection
- AI-Generated Insights
- Version Timeline Analysis
- Future Improvement Prediction

## Tech Stack
Python, Streamlit, LangChain, FAISS, Ollama

## Project Idea
The system compares old and new code versions and generates intelligent insights about maintainability, scalability, and architectural improvements.
